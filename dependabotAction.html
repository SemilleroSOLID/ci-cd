Configuración de Dependabot

Para mantener actualizadas las dependencias de este proyecto, hemos configurado Dependabot. A continuación, se detallan los pasos básicos para revisar y fusionar las actualizaciones:

  1. Revisa las solicitudes de extracción (pull requests) abiertas en la pestaña "Pull requests" de este repositorio.
  2. Busca las solicitudes de extracción creadas por Dependabot.
  3. Revisa los cambios propuestos por Dependabot y asegúrate de que no haya conflictos con el código existente.
  4. Si todo parece correcto, puedes fusionar la solicitud de extracción para aplicar las actualizaciones de dependencias.
