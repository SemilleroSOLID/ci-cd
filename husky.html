Configuración Manual de Husky para Node.js

Para configurar Husky manualmente en tu proyecto Node.js, sigue estos pasos:

  1. Instalar Husky: Ejecuta el siguiente comando en la terminal:
    2. npm install husky --save-dev
  2. Habilitar los ganchos de Git: Ejecuta el siguiente comando para habilitar automáticamente los ganchos de Git después de la instalación:
    3. npx husky install
  3. Configurar el script "prepare" en package.json: Edita tu archivo package.json para incluir el siguiente script:
    4. "scripts": {
  "prepare": "husky install"
}
  4. Crear un gancho: Para agregar un comando a un gancho o crear uno nuevo, utiliza husky add [cmd] asegúrate de ejecutar husky install antes antes
    5. npx husky add .husky/pre-commit "npm test"
  5. Agregar el gancho a Git: Ejecuta el siguiente comando para agregar el nuevo gancho a Git:
    6. git add .husky/pre-commit
  6. Intentar realizar un commit: Ejecuta el siguiente comando para realizar un commit y probar la configuración de Husky:
    7. git commit -m "Keep calm and commit"
  7. Las pruebas que realizamos fueron en un entorno local, para controlar el commit

    8. Si el comando npm test falla, tu commit se abortará automáticamente.
